# local — 커밋하지 않는 도면

인터넷 교보재·타사 도면처럼 **재배포할 수 없는 자료**를 여기에 둔다.
이 폴더는 `.gitignore`에 등재돼 있어 실수로 커밋되지 않는다 (이 README만 예외).

여기 도면으로 얻은 것은 **측정 수치와 발견한 결함**이지 파일이 아니다.
실증 앵커는 `docs/VALIDATION_EVIDENCE.md`에 기록하고, 재현이 필요하면
같은 성질의 비민감 도면을 `synthetic/` 또는 재배포 가능한 `public/`에 남긴다.

이렇게 나누는 이유는 저작권만이 아니다. 실제 프로젝트 도면에는 발주처명·현장
주소·설계사 인장이 박혀 있어 영업비밀·개인정보 문제가 따라온다. 로컬에서 읽고
버리는 것과 공개 리포에 박아 재배포하는 것은 다른 행위다.

회사 도면은 공개 자료와 같은 폴더에 섞지 않는다. 로컬 검증 결과를 문서화할 때도 파일명, 발주처명, 주소, 인장과 원본 이미지를 외부 문서에 복제하지 않는다.
