# public — 공공·퍼블릭도메인 도면

한전 표준 결선도, 조달청 공개 입찰 첨부 도면, 공공기관 교육 교재, 미 연방정부
저작물(PD), 오픈소스 CAD 테스트 픽스처 등 **재배포 가능한 자료**를 둔다.

각 도면에는 `<id>.label.json`을 함께 두고 `source`·`license` 필드를 반드시 채운다.
출처를 적을 수 없는 자료는 여기가 아니라 `local/`로 간다.

여기 자료의 값어치는 전기 내용이 아니라 **CAD 방언의 다양성**에 있다. 파서를
깨뜨리는 건 회로의 복잡도가 아니라 파일이 어떻게 작성됐는가다.
