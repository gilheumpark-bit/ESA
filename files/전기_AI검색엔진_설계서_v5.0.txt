1. 프로젝트 개요
==================================================

한 줄 정의:
전기 엔지니어링 특화 글로벌 AI 플랫폼.
전 세계 전기 규격 (KEC/IEC/NEC/JIS/GB/AS-NZS/EN 등) 완전 지원 +
글로벌 논문/규격/뉴스 실시간 수집 + 할루시네이션 없는 출처 기반 답변 +
영수증 계산기 + 날짜 박제 + 협업 + 현장 모바일 + 법적 공증 +
해외 플랜트 규격 자동 변환까지 원스톱 엔지니어링 플랫폼.

시장 기회:
  가트너 예측: 2026년까지 전통 검색엔진 볼륨 25% 감소
  버티컬 LLM 시장: 2025년 2.9B달러 -> 2033년 18.7B달러 (CAGR 26%)
  전기 특화 버티컬 검색: 전 세계 공백 상태
  해외 플랜트 EPC 시장: 한국 3대 EPC사 연간 수십조 규모
  글로벌 전기 엔지니어: 수백만 명 잠재 사용자

2. 핵심 철학 및 원칙
==================================================

원칙 1: 할루시네이션 제로
  AI = 수집된 문서 기반 요약만 수행
  출처 없는 답변 = 시스템 레벨에서 차단
  수치/규격 = 계산 엔진이 직접 연산

원칙 2: 날짜 박제
  모든 데이터에 타임스탬프 필수
  원문 발행일 / 수집일 / 검증일 / 규격 버전 / 계산 실행 시각 / 엔진 버전
  언제 기준인지 모르면 현장에서 못 쓴다

원칙 3: 영수증 투명성
  계산 과정 전체 공개
  출처 논문/규격 조항까지 명시
  AI가 계산했다 X -> 엔진이 직접 돌린 결과

원칙 4: 법적 명확성
  모든 산출물에 면책 조항 자동 삽입
  공증은 시점 증명이지 결과 보증이 아님을 명시

원칙 5: 글로벌 규격 동등성 (v5.0 추가)
  어떤 국가 규격으로 질문해도 답변 가능
  국가 간 규격 자동 변환 및 차이점 명시
  단위 체계 자동 변환 (mm2 vs AWG, kV vs V 등)

원칙 6: 도메인 플러그인
  전기 -> 기계 -> 토목 -> 화학 -> 전체 공학
  코어 건드리지 않고 도메인만 추가

원칙 7: Hybrid BYOK
  Free: 플랫폼 기본 소형 LLM 제공
  Pro 이상: 사용자 본인 API 키
  API 키는 암호화 저장, 로그 기록 안 함

원칙 8: 법적 방패
  날짜 박제 + 블록체인 공증 (익명화 적용)
  = 당시 현행 규격 기준으로 작성됨을 타임스탬프로 영구 증명

11. 해외 플랜트 규격 자동 변환 - NEW v5.0
==================================================

배경:
  한국 EPC사 해외 플랜트 수주 시
  설계는 KEC 기준 -> 현지 NEC/IEC/AS-NZS 기준으로 변환 필요
  이 작업이 현재 수작업으로 오류 발생 + 시간 낭비

기능 개요:
  소스 규격 선택 (KEC)
  + 타겟 규격 선택 (NEC)
  + 계산 결과
  -> 자동 변환 + 차이점 리포트

변환 예시 1: KEC -> NEC 케이블 선정

  입력: KEC 기준 95mm2 케이블
  변환:
    95mm2 = 3/0 AWG (약 85mm2 상당)
    또는 4/0 AWG (107mm2 상당)
    NEC 허용전류 기준 재계산 필요
    온도 보정 계수 차이 적용
  출력:
    KEC 기준: 95mm2 CV 케이블
    NEC 기준: 4/0 AWG THWN-2
    차이점: NEC는 AWG 체계, 온도 기준 40도C vs 60도C 상이
    영수증 생성 (두 기준 병기)

변환 예시 2: KEC -> NEC 전압강하

  입력: KEC 기준 2.3% 전압강하 (간선 3% 이내 적합)
  변환:
    NEC에는 전압강하 의무 기준 없음 (FPN 권고만)
    NEC FPN: 5% 권고
    현재 2.3% -> NEC 기준으로도 문제없음
  출력:
    KEC 기준: 2.3% (적합, 3% 이내)
    NEC 기준: 2.3% (적합, 5% 이내 권고)
    결론: 두 규격 모두 만족

변환 예시 3: KEC -> JIS 변환

  입력: KEC 기준 22.9kV 수변전설비
  변환:
    일본 배전전압: 6.6kV (다름!)
    변압기 2차측 전압: 한국 380V vs 일본 400V
    계통 구성 재검토 필요
  출력:
    규격 충돌 경고!
    일본 배전전압 체계가 달라 설비 재설계 필요
    JIS 내선규정 기준 재계산 권장

변환 엔진 구조:

  StandardConversionEngine:
    source_standard: KEC
    target_standard: NEC
    concept: cable_sizing
    input_value: 95 mm2
    -> lookup_global_mapping()
    -> apply_unit_conversion()
    -> apply_local_corrections()
    -> generate_conversion_receipt()
    -> generate_diff_report()

변환 지원 항목:
  케이블 단면적 (mm2 <-> AWG <-> kcmil)
  허용전류 (온도 기준 보정)
  전압강하 허용 기준 비교
  차단기 정격 (IEC <-> UL/ANSI)
  접지 저항 기준
  변압기 임피던스 기준
  보호 협조 기준

변환 불가 알림:
  전압 체계 자체가 다른 경우 (22.9kV -> 6.6kV)
  -> 단순 변환 불가, 재설계 필요 경고
  규격 간 개념 자체가 없는 경우
  -> NEC에 해당 조항 없음 명시

EPC 특화 기능:
  프로젝트 단위 규격 변환 이력 관리
  발주처 규격 요구사항 등록
    사우디 아람코 -> NEC + SAES 기준
    UAE ADNOC -> IEC + ADCO 기준
  변환 결과 일괄 PDF/Excel Export
  감리관 제출용 규격 비교 보고서 자동 생성

14. 에이전트 워크플로우
==================================================

개념:
  질문 -> 답변 -> 다음 액션 -> 결과물
  목표: 엔지니어 업무 흐름 플랫폼 안에서 완결

글로벌 워크플로우 예시 (해외 플랜트):

  입력: "사우디 아람코 플랜트 전기 설계
         KEC 기준 설계본을 NEC + SAES 기준으로 변환해줘"

  Step 1: KEC 기준 원본 계산서 불러오기
  Step 2: 타겟 규격 확인 (NEC + SAES 사우디 기준)
  Step 3: 국가 전압 체계 확인
    한국 22.9kV -> 사우디 34.5kV / 13.8kV
    전압 체계 차이 경고 발생
  Step 4: 변환 가능 항목 자동 변환
    케이블: mm2 -> AWG
    허용전류: 온도 기준 보정 (40도C -> 30도C 기준)
    차단기: IEC 정격 -> ANSI/UL 정격
  Step 5: 변환 불가 항목 목록
    전압 체계 재설계 필요 항목
  Step 6: 규격 비교 리포트 생성
    KEC vs NEC 항목별 비교
    준수 여부 체크리스트
  Step 7: Export
    PDF 비교 보고서 (영문 + 면책 조항)
    Excel 계산서 (NEC 기준 수식)

국내 워크플로우 예시 (수변전설비):

  입력: "22.9kV 수변전설비 설계
         (부하 500kW, 역률 0.9, 케이블 200m)"

  Step 1: 변압기 용량 계산 -> 영수증 #001
  Step 2: KEC 조항 확인 (2024년판)
  Step 3: 케이블 선정 -> 영수증 #002
  Step 4: 전압강하 검증 -> 영수증 #003
  Step 5: 차단기 용량 -> 영수증 #004
  Step 6: 체크리스트 생성
  Step 7: Export (PDF + Excel + 블록체인 공증)

26. 블록체인 공증 (익명화)
==================================================

필요성:
  전기 설계 계산서 법적 보존 10년 이상

GDPR/개인정보 해결:
  공증 전 민감 정보 익명화
    프로젝트명 / 담당자명 / 기업명 / 지역명 -> 해시 대체
  원본: 플랫폼 DB (삭제 가능)
  IPFS: 익명화 버전만 (삭제 불가하나 개인정보 없음)

공증 프로세스:
  영수증 완성
  -> 면책 조항 동의
  -> 익명화 처리
  -> SHA-256 해시
  -> IPFS 저장
  -> 블록체인 타임스탬프
  -> 공증 배지 + 검증 URL

34. 정확도 목표
==================================================

계산기 (영수증): 99.5% 이상
  Python 엔진 직접 연산

AI 검색 (RAG): 85~92%
  도메인 특화

글로벌 규격 비교 정확도 (v5.0): 90~95%
  수동 검증된 규격 매핑 DB 기반
  규격 개정 시 즉시 업데이트

규격 변환 정확도 (v5.0): 88~93%
  단순 변환 (mm2 <-> AWG): 99%+
  복합 변환 (허용전류 + 온도보정): 90%
  규격 체계 자체 다른 경우: 변환 불가 명시

Vision-LLM PDF 파싱: 88~95%

뉴스 번역/요약: 88~93%

자격증/일정: 95% 이상

OCR 명판 인식: 90% 이상 (다국어 포함)

정확도 향상 전략:
  단기: A/B 테스트, 수치 범위 검증, 규격 매핑 전문가 검수
  중기: 피드백 루프, 커뮤니티 버그 리포트
  장기: 전기 특화 LLM 파인튜닝, 지식 그래프 고도화

핵심 차별점 요약 (v5.0 최종)
==================================================

1. 영수증 시스템 (독보적)
   계산 과정 전체 공개 -> 특허 출원 검토

2. 날짜 박제 (법적 방패)
   언제 어떤 규격/국가 기준인지 타임스탬프 영구 증명

3. 글로벌 규격 완전 커버 (v5.0)
   KEC / IEC / NEC / CEC / EN / DIN VDE / BS 7671
   NF C / JIS / 내선규정 / GB / DL / AS-NZS
   전 세계 전기 규격 단일 플랫폼

4. 해외 플랜트 규격 자동 변환 (v5.0)
   KEC -> NEC / IEC / JIS 원클릭 변환
   EPC사 해외 프로젝트 수작업 제거

5. 국가별 전압/전기 체계 DB (v5.0)
   계산기 국가 선택 -> 표준전압/주파수/규격 자동 적용

6. 단위 변환 엔진 (v5.0)
   mm2 <-> AWG <-> kcmil 자동 변환
   IEC <-> ANSI 정격 변환

7. 면책 조항 자동화 (다국어)
   한국어/영어/일본어/중국어 면책 조항 자동 삽입

8. Vision-LLM PDF 파싱
   IEC/IEEE/NEC/JIS/GB 규격서 표/수식까지 완전 색인

9. Hybrid BYOK
   기본 소형 LLM 탑재로 진입 장벽 제거

10. 엑셀 수식 Export
    수식 살아있는 계산서 -> 감리/관공서 제출

11. 블록체인 공증 + 익명화 (GDPR 대응)
    위변조 불가 공증 + 개인정보 보호 동시

12. 에이전트 워크플로우
    검색 -> 계산 -> 변환 -> Export -> 공증 원스톱

13. 전기 x AI 교차 도메인
    전 세계 없는 기능

14. 규격 개정 추적 (전 세계)
    KEC/IEC/NEC/JIS/GB 등 개정 감지 -> 즉시 알림
