## 1. 시작하기

### 접속
- URL: `https://esva.engineer`
- 로그인 없이 모든 계산기를 사용할 수 있습니다.
- Google 계정으로 로그인하면 이력 저장, 영수증 관리, 프로젝트 기능을 사용할 수 있습니다.

### 로그인
1. 우측 상단 **로그인** 버튼 클릭
2. **Google로 로그인** 버튼 클릭
3. Google 계정 선택 → 자동 로그인
4. 로그인 없이 사용하려면 **건너뛰기** 클릭

### 메인 화면 구성
| 영역 | 설명 |
|------|------|
| 검색바 | 전기 관련 질문을 자연어로 입력 |
| 전기 계산기 | 52개 전문 계산기 바로가기 |

| 기준서 | 90조 기준서 직접 조회 |

## 2. 전기 계산기

### 지원 계산기 (52개, 12개 분야)

| 분야 | 계산기 예시 | 수 |
|------|-----------|---|
| 전력기초 | 단상/3상 전력, 역률, 무효전력 보상 | 7 |
| 전압강하 | 기본, 3상, 임피던스, 부스바, 국가별 비교 | 5 |
| 케이블 | 사이징, AWG↔mm² 변환, 허용전류 비교 | 4 |
| 변압기 | 용량, 손실, 효율, 임피던스, 돌입전류 | 6 |
| 보호협조 | 단락전류, 차단기 선정, 지락전류, RCD, **아크플래시(IEEE 1584)** | 6 |
| 접지 | 접지저항, 접지도체, 등전위본딩, 피뢰 | 4 |
| 전동기 | 용량, 기동전류, 효율, 인버터, 역률보상, 제동저항 | 6 |
| 신재생/ESS | 태양광 발전, 배터리, PCS, 계통연계 | 5 |
| 수변전 | 수변전용량, CT/VT 선정, 서지어레스터 | 4 |
| 조명 | 조도계산, 에너지절감, UPS, 비상발전 | 4 |
| 글로벌 | 온도보정, 주파수비교, AWG변환, NEC부하계산 | 5 |

### 사용 방법
1. `/calc` 페이지에서 원하는 계산기 선택
2. 입력값 입력 (전압, 전류, 길이, 케이블 규격 등)
3. **계산** 버튼 클릭
4. 결과 확인:
   - **게이지 차트**: 기준값 대비 시각적 표시 (PASS/FAIL)
   - **단계별 풀이**: 수식 전개 과정
   - **적용 기준**: KEC/NEC/IEC 조항 번호
   - **영수증**: 날짜, 모델, 해시 포함 검증 기록

### 결과 내보내기
- **PDF 다운로드**: 계산 과정 + 기준서 참조 포함
- **Excel 다운로드**: 데이터 재활용 가능
- **공유**: 링크 복사로 결과 공유

### 정확도
- 모든 계산기는 **±0.01%** 정확도로 검증됨
- 참조값: 공학 핸드북 + 상용 소프트웨어 대조
- 323개 자동화 테스트 통과

## 3. AI 법규 검색

### 지원 기준서
| 기준서 | 국가 | 조항 수 | 판 |
|--------|------|---------|---|
| KEC | 한국 | 61조 | 2021 |
| NEC | 미국 | 19조 | 2023 |
| IEC 60364 | 국제 | 10조 | 2017 |

### 사용 방법
1. `/search` 페이지 또는 메인 검색바에 질문 입력
2. 예시 질문:
   - "KEC 232.52 전압강하 기준"
   - "3상 380V 100A 50m 전압강하 계산해줘"
   - "NEC 310.16 허용전류 테이블"
   - "한국과 미국 전압강하 기준 비교"
3. 결과:
   - **검색 결과**: 관련 조항 + 출처 표시
   - **지식 패널**: IEC 60050 용어 정의
   - **관련 계산기**: 자동 추천
   - **교차참조**: KEC↔NEC↔IEC 등가 조항

### BYOK (자체 API 키)
- 기본: 일일 500,000 토큰 무료
- BYOK: 설정 → API 키 등록 → 무제한
- 지원 프로바이더: Gemini, OpenAI, Claude, Groq, Mistral

### 안전 장치
- **AI는 수치를 추정하지 않습니다** — 모든 수치는 계산기 Tool로 산출
- **출처 없는 답변은 차단됩니다** — 기준서 조항 없이 숫자를 생성하면 필터링
- **아크플래시/기동전류 추정 금지** — 전용 계산기 사용 필수

## 4. 도면 분석 (SLD)

### 지원 형식
| 형식 | 방식 | API 키 |
|------|------|--------|
| 이미지 (PNG/JPG) | AI Vision (Gemini/GPT-4.1) | BYOK 필수 |
| DXF (AutoCAD) | 벡터 파싱 (100% 정확) | 불필요 |
| PDF | 벡터 추출 | 불필요 |

### 사용 방법
1. `/tools/sld` 페이지 접속
2. 탭 선택: **이미지 AI** / **DXF 벡터** / **PDF 벡터**
3. 파일 업로드 (이미지 최대 20MB, DXF 최대 50MB, PDF 최대 100MB)
4. 결과:
   - **컴포넌트 목록**: 변압기, 차단기, 전동기, 부하 등 인식
   - **연결 관계**: 어디서 어디로, 케이블 사양
   - **토폴로지 검증**: 격리 노드, 순환 경로 등 이상 탐지
   - **계산 체인 추천**: "전압강하 계산 필요", "차단기 선정 필요" 등

### 4-Team 설계 리뷰
도면 업로드 시 **4개 전문 팀**이 병렬로 분석합니다:

| 팀 | 역할 |
|----|------|
| 계통도팀 (TEAM-SLD) | 계통도 분석 → 토폴로지 → 계산 |
| 평면도팀 (TEAM-LAYOUT) | 배선 경로 → 전선관 → 거리 산출 |
| 규정질의팀 (TEAM-STD) | KEC/NEC/IEC 조항 대조 |
| 합의+출력팀 (TEAM-CONSENSUS) | 팀 간 토론 → 검증 마킹 → 보고서 |

## 5. 기준서 조회

### 사용 방법
1. `/standards` 페이지 접속
2. 좌측에서 기준서 선택 (KEC, NEC, IEC)
3. 조항 클릭 → 우측 상세 패널:
   - 조항 원문 (조건 트리)
   - 교차참조 (KEC↔NEC↔IEC 등가)
   - 허용전류 빠른 조회
   - 관련 계산기 링크

### 허용전류 빠른 조회
1. 기준서 페이지에서 **허용전류** 탭 선택
2. 전선 규격 (mm²), 도체 재질 (Cu/Al), 절연 종류 선택
3. KEC 기준 허용전류 즉시 표시

## 6. 검증 보고서

### 보고서 구성
`/report/[id]` 페이지에서 확인:

| 구역 | 내용 |
|------|------|
| **게이지 대시보드** | 각 검증 항목의 PASS/FAIL 게이지 차트 |
| **도면 마킹** | 도면 위 빨강(오류)/노랑(경고)/초록(적합) 핀 |
| **ESVA Verified 배지** | 등급(A+~F), 점수, 판정(적합/조건부/부적합) |
| **검증 마킹 목록** | 모든 항목 상세 (위치, 계산값, 기준, 수정 제안) |
| **팀별 결과** | 4개 팀의 분석 결과 |
| **토론 기록** | 팀 간 불일치 → 합의 과정 |

### PDF 내보내기
1. 보고서 하단 **PDF 다운로드** 클릭
2. 인쇄 대화상자에서 **PDF로 저장** 선택
3. 포함 내용:
   - ESVA 워터마크
   - 3칸 서명란 (설계자 / 검토자 / PE 승인자)
   - PE급 면책조항
   - SHA-256 해시 추적

## 8. BYOK (API 키 설정)

### API 키 등록 방법
1. `/settings` 페이지 → **BYOK API 키 관리**
2. 사용할 AI 프로바이더 선택:

| 프로바이더 | 추천 모델 | 비용 |
|-----------|----------|------|
| Google Gemini | Gemini 2.5 Flash | 무료 (일일 한도) |
| OpenAI | GPT-4.1 Mini | $0.40/1M 토큰 |
| Anthropic | Claude Sonnet 4 | $3.00/1M 토큰 |
| Groq | Llama 4 Scout | 무료 |

3. API 키 입력 → **저장**
4. 키는 브라우저에만 저장되며, ESVA 서버에 전송되지 않습니다.

### 보안
- API 키는 AES-GCM 암호화 후 localStorage에 저장
- 서버 전송 시 요청 헤더로만 전달 (로그 미기록)
- ESVA는 키를 저장하지 않습니다

## 9. 계산 이력 / 영수증

### 이력 관리
1. `/history` 페이지에서 모든 계산 이력 조회
2. 필터: 카테고리, 판정(PASS/FAIL), 날짜 범위
3. CSV 내보내기: 전체 이력을 스프레드시트로 다운로드

### 영수증 (Receipt)
모든 계산 결과에는 검증 가능한 영수증이 포함됩니다:
- **고유 ID**: UUID v4
- **타임스탬프**: 계산 시각 (UTC)
- **SHA-256 해시**: 입력+결과의 무결성 검증
- **적용 기준**: KEC/NEC/IEC 조항 번호 + 판
- **면책조항**: PE 검토 권고 포함

## 10. 자주 묻는 질문

### Q: 계산 결과를 신뢰할 수 있나요?
A: 모든 계산기는 ±0.01% 정확도로 검증되며, 323개 자동화 테스트를 통과합니다. 단, **최종 설계에는 반드시 PE(기술사) 검토가 필요합니다.** 아크플래시, 단락전류 등 안전 관련 계산은 전문 소프트웨어(ETAP, SKM)로 재검증하세요.

### Q: 무료로 사용할 수 있나요?
A: 네. 모든 계산기는 무료입니다. AI 검색은 일일 500,000 토큰까지 무료이며, BYOK로 자체 API 키를 등록하면 무제한 사용 가능합니다.

### Q: 데이터는 어디에 저장되나요?
A: 로그인하지 않으면 브라우저(sessionStorage)에만 저장됩니다. 로그인하면 Supabase(클라우드 DB)에 영구 저장됩니다. API 키는 서버에 저장되지 않습니다.

### Q: 어떤 기준서를 지원하나요?
A: KEC 2021 (한국), NEC 2023 (미국), IEC 60364 (국제). 총 90조입니다.

### Q: 오프라인에서 사용할 수 있나요?
A: 계산기는 순수 함수로 오프라인 사용이 가능합니다 (PWA 지원). AI 검색과 도면 분석은 인터넷 연결이 필요합니다.

### Q: 버그를 발견하면?
A: `/settings` → **문의** 또는 이메일로 신고해주세요. 기준서 조항 오류는 즉시 수정됩니다.
